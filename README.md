# CM English

## Como rodar

Requisitos: Node.js + npm (ou Bun).

```sh
npm install
npm run dev
```

Abra o endereco mostrado no terminal (normalmente http://localhost:5173).

## Scripts

- `npm run dev` - servidor de desenvolvimento
- `npm run build` - build de producao
- `npm run preview` - preview do build
- `npm run lint` - lint
